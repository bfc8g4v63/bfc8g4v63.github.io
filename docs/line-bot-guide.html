LINE 群組機器人
設定教學

程式端已準備好。只需建立一個 LINE 官方帳號、取得兩個私密憑證，再把機器人邀進家族群組。

一、建立 LINE 官方帳號

  1. 登入 LINE Official Account Manager，建立官方帳號，例如「好日子家庭提醒」。
  2. 在官方帳號後台啟用 Messaging API，並選擇或建立 Provider。
  3. 進入 LINE Developers Console 的 Messaging API channel。

二、取得兩個私密憑證

  1. 在 Basic settings 複製 Channel secret。
  2. 在 Messaging API 頁籤簽發並複製 Channel access token。
  3. 把兩者設到後端環境變數 LINE_CHANNEL_SECRET 與 LINE_CHANNEL_ACCESS_TOKEN。不要放進 GitHub 程式碼或群組訊息。

這兩個值等同機器人的密碼；網站訪客與 GitHub 公開 repo 都不會看到它們。

三、設定 Webhook

在 Messaging API 頁籤把 Webhook URL 設為：

https://good-days-family-events.x0925234139.chatgpt.site/api/line/webhook

  1. 開啟 Use webhook。
  2. 按 Verify，應顯示成功。
  3. 開啟 Allow bot to join group chats。

四、把活動綁定到家族群組

  1. 把剛建立的 LINE 官方帳號加為好友，再邀請進家族群組。
  2. 回到好日子網站，點活動的「管理」，輸入活動管理碼。
  3. 在 LINE 區塊點「產生群組綁定碼」。
  4. 在家族群組輸入 綁定 123456（換成畫面上的六碼）。
  5. 看到「綁定成功」後，可回管理後台傳送測試提醒。

提醒規則

預設在活動前 7 天與 1 天提醒，也可加開活動前 2 小時。排程每 15 分鐘檢查一次，因此實際傳送時間可能前後差約 15 分鐘。

LINE 群組同時間只能加入一個官方帳號；提醒訊息會計入該官方帳號的 Messaging API 訊息用量。
