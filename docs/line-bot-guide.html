LINE 小幫手
使用說明

所有人都可以加入好日子 LINE 小幫手。把小幫手邀請進活動群組後，就能在群組內查看目前的報名資訊。

一、加入好日子 LINE 小幫手

先把好日子 LINE 小幫手加為好友。

加入 LINE 小幫手

二、邀請到活動群組

將好日子 LINE 小幫手邀請到活動所在的 LINE 群組。活動管理者完成群組綁定後，所有群組成員都可以使用小幫手。

三、輸入群組指令

在已綁定的 LINE 群組中，所有成員都可以使用以下指令：

活動
小幫手會廣播這筆活動的分享連結與 QR Code，方便其他人掃描查看和報名。
原神啟動
小幫手會廣播目前報名名單，包含姓名、人數、飲食、備註，以及總參加人數。
