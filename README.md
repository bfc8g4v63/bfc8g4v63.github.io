# 好日子 x 相聚

讓每一次相聚，更容易。

「好日子 x 相聚」是一個為家人、朋友與社群設計的活動建立與報名網站。介面以大字、清楚步驟與跨裝置瀏覽為原則，讓長輩也能輕鬆查看活動、填寫出席狀況與收到 LINE 提醒。

網站網址：https://bfc8g4v63.github.io/

## 適合的使用情境

- 家庭聚餐、節慶團圓與親友相約
- 約會、生日、旅行與同學會
- 社區活動、小型課程與志工聚會

## 目前功能

### 建立與管理相聚

- 建立活動：日期、時間、地點、說明、聯絡人與人數上限
- 管理碼保護：建立者可查看管理後台、修改內容與取消活動
- 管理後台可查看出席人數、回覆清單、飲食需求與備註
- 活動聯絡電話僅供管理者查看，不會公開顯示在活動頁面

### 報名與隱私

- 參加者可填寫姓名、出席狀態、同行人數、飲食需求與備註
- 公開頁面只顯示統計人數，不公開參加者名單
- 支援三種活動可見範圍：
  - **公開**：顯示在首頁，適合社區或公開招生。
  - **不公開**：不列在首頁，持有分享連結的人可以參加。
  - **私人**：不列在首頁，需同時持有分享連結與參加碼。
- 每個不公開或私人活動都有專屬分享網址與 QR Code，方便用 LINE 傳給親友。

### LINE 小幫手

- 可將「好日子－相聚小幫手」加入活動群組並完成綁定。
- 依設定在活動前 7 天、1 天或 2 小時發送群組提醒。
- 機器人不會回覆一般群組聊天，避免干擾大家對話。
- 首頁與活動頁皆提供加入 LINE 好友的引導。

### 使用體驗

- 支援 iPhone、iPad、Android、Windows 與一般瀏覽器。
- 首頁可瀏覽公開活動；專屬活動可由分享連結直接開啟。
- 採用響應式版面與較易閱讀的字級，適合各年齡使用。
