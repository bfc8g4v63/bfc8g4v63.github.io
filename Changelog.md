# Changelog

## v1.2.4 · 2026-07-22

### 管理功能修正

- 修正活動修改、取消與永久刪除未正確帶入建立者驗證資料，導致已驗證管理者仍被拒絕的問題。

## v1.2.3 · 2026-07-22

### 前端更新

- 更新網站快取與程式檔版本，確保活動管理表單會載入最新內容，包括「永久刪除」與出席名單可見性設定。

本檔案記錄「好日子 x 相聚」的重要功能與內容變更。

## v1.2.2 · 2026-07-21

### 報名人數上限

- 人數上限改由後端強制執行，新增報名、增加人數與同時送出的回覆都不能超過上限。
- 已報名者在額滿後，仍可使用相同姓名更新內容、減少人數或改為不參加。
- 首頁與活動頁會在額滿時顯示「活動已額滿」。

## v1.2.1 · 2026-07-21

### 建立者與活動管理

- 新增建立者管理連結與找回活動流程。
- 舊活動可沿用既有聯絡人姓名與 4 位數管理碼找回。
- 新增活動公開範圍、名單顯示方式與活動取消／永久刪除管理。

### LINE 與體驗

- LINE 群組可使用「活動」分享連結與 QR Code，以及「原神啟動」廣播報名摘要。
- 首頁新增 LINE 小幫手引導、活動公開方式說明與網站瀏覽人次。
- 修正活動連結逾時提示、找回視窗錯誤顯示與頁尾瀏覽人次版面。

## 2026-07-20

### 品牌與內容

- 品牌名稱調整為「好日子 x 相聚」。
- 網站標語調整為「讓每一次相聚，更容易。」
- 活動相關用語由「家族活動」調整為「相聚」。
- 首頁與活動頁加入 LINE 好友引導。
- 首頁新增「活動公開方式」說明，協助建立者選擇公開、不公開或私人參加碼。

### 活動與隱私

- 新增公開、不公開與私人三種活動可見範圍。
- 不公開及私人活動提供專屬分享連結與 QR Code。
- 私人活動需輸入參加碼才能查看及填寫回覆。
- 建立者可在管理後台調整既有活動的可見範圍。
- 活動聯絡電話改為僅管理者可查看。
- 報名者的姓名、飲食需求與備註僅在管理後台顯示。

### 管理與 LINE 提醒

- 新增受管理碼保護的活動管理後台。
- 管理者可查看出席統計與私密報名清單。
- 新增 LINE 群組綁定功能。
- 已綁定的 LINE 群組輸入「活動」，小幫手會回覆該活動的分享連結與 QR Code。
- 「活動」指令同步顯示日期、時間、餐廳／地點與活動說明，方便公告地點調整。
- 可設定活動前 7 天、1 天與 2 小時的 LINE 群組提醒。
- LINE 機器人僅處理加入與綁定，不再回覆一般群組訊息。

### 發布

- 公開網站遷移至 GitHub Pages。
- 後端 API 與資料庫持續由網站服務提供。
